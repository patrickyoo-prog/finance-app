# 💰 나의 가계부

아이폰에서 간편하게 사용하는 PWA 기반 가계부 입력 앱입니다.

## 🎯 기능

- ✅ **간단한 입력**: 날짜, 금액, 카테고리, 메모만 입력
- 📱 **PWA 지원**: 아이폰 홈화면에 앱처럼 설치 가능
- 💾 **자동 저장**: 입력 즉시 노션에 저장
- 📊 **카테고리 분류**: 8가지 기본 카테고리 + 자유 입력
- 🔄 **자동 동기화**: 매월 말일에 분석 보고서 자동 생성 및 이메일 전송
- 🌙 **다크모드**: 시스템 설정에 따른 자동 적용

## 📲 설치 방법 (아이폰)

### 1단계: 앱 열기
1. Safari에서 https://patrickdongkiyoo.github.io/finance-app/ 접속
2. 주소창 하단의 **공유** 버튼 클릭

### 2단계: 홈화면에 추가
1. **공유** 메뉴에서 **"홈 화면에 추가"** 선택
2. 앱 이름 확인 후 **"추가"** 클릭
3. 완료!

이제 아이폰 홈화면에서 "나의 가계부" 아이콘으로 앱을 실행할 수 있습니다.

## 💡 사용 방법

### 기본 입력
1. **거래 유형**: 💸 지출 또는 💵 수입 선택
2. **날짜**: 자동으로 오늘 날짜 설정됨 (변경 가능)
3. **금액**: 거래 금액 입력 (숫자만)
4. **카테고리**: 드롭다운에서 선택
   - 🍽️ 식비
   - 🚗 교통비
   - 🛍️ 쇼핑
   - 🏥 의료
   - 📚 교육비
   - 🎬 취미
   - 🛒 마트 장보기
   - 📌 기타
5. **메모**: (선택) 추가 정보 입력
6. **💾 입력** 버튼 클릭

### 자동 저장
- 입력 완료 시 즉시 노션 가계부에 저장됨
- 최근 입력 항목이 하단에 표시됨

## 🔧 기술 구성

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Vercel Serverless Functions (4단계에서 설정)
- **Database**: Notion API
- **Automation**: GitHub Actions (매월 말일 실행)
- **Email**: Gmail SMTP (보고서 전송)

## 📞 문제 해결

### Q: 입력이 저장되지 않습니다
A: 네트워크 연결을 확인하고, 브라우저 콘솔(F12)에서 오류 메시지를 확인하세요.

### Q: 홈화면에 추가되지 않습니다
A: Safari를 사용하고 있는지 확인하세요. Chrome에서는 공유 → "Chrome 메뉴에 추가"로 추가 가능합니다.

### Q: 카테고리 수정은 어떻게 하나요?
A: GitHub의 `index.html` 파일을 수정해서 옵션을 추가/삭제하면 됩니다.
